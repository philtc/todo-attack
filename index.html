Keyboard Shortcuts

  • Ctrl+S - Save file
  • Ctrl+O - Load file
  • Ctrl+T - Toggle task status
  • Ctrl+D - Add today's date
  • Ctrl+N - Insert new task
  • Ctrl+H - Toggle help

Todo Syntax

  • [ ] - Pending task
  • [/] - In progress
  • [x] - Completed
  • +tag - Add tags
  • due:YYYY-MM-DD - Due date
  • (a) - High priority
